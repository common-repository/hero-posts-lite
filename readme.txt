= Awesome Features =

HeroPosts Lite Plugin by Bold Themes is a very flexible tool that gives you great possibilities for customization. Here are some of the most prominent features of HeroPosts Lite:

* Variable number of posts 
* Post filters (category, author, date)
* Showing posts in a slider
* Accent color

== Installation ==

Use WordPress Add New Plugin feature (Plugins panel) and search for HeroPosts Lite. Click install and then activate (once the plugin has been installed). 

No 3rd party services needed.
